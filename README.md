# ChannelAdam.Nancy.Soap

A .NET library of helpers and extensions for working with SOAP payloads in the Nancy web framework.

Targets:

- .NET 5.0
- .NET Standard 2.0

## Usage Documentation

Please see https://devzone.channeladam.com/library/channeladam-nancy-soap/
